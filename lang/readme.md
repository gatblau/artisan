# Artisan Language Dictionaries

This folder contains dictionaries to internationalise Artisan in the following languages:

| code | ISO 639-1 language name |
|---|---|
| de | [german](de.i18n) |
| en | english (default) |
| es | [spanish](es.i18n) |
| fr | [french](fr.i18n) |
| hi | [hindi](hi.i18n) |
| zh | [chinese](zh.i18n) |
